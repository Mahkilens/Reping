# Reping – Full-Stack Gym Consistency Application

Reping is a full-stack fitness tracking platform designed to help users build consistency through streak tracking, gamification, and structured progress monitoring.

## 🚀 Overview

Reping allows users to:
- Track daily gym attendance
- Maintain streaks with one check-in per day validation
- View progress through dynamic dashboards
- Earn XP through a gamified progression system

The application is built with a focus on backend logic, data integrity, and scalable relational modeling.
